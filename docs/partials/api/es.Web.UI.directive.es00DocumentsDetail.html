es00DocumentsDetail
directive in module es.Web.UI

Description

TBD This directive is responsible to render the html for the presentation of the ES00Documents as a detail of a kendo-grid

Dependencies

  • esWebApi

    Entersoft AngularJS WEB API for Entersoft Application Server

  • esUIHelper
  • $log

Usage

as element:
<es00-documents-detail
       [es-document-grid-options="{object}"]
       [es-master-row-field="{string}"]>
</es00-documents-detail>
as attribute
<ANY es00-documents-detail
     [es-document-grid-options="{object}"]
     [es-master-row-field="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
esDocumentGridOptions
(optional)
object

A subset or full set of esGridOptions for the kendo-grid that will show the ES00Documents. The ES00Documents kendo-grid will be initialized by the merge of the PublicQueryInfo gridoptions as retrieved for the GroupID = "ESGOCompany" and FilterID = "ES00DocumentsDetails" public query.

esMasterRowField
(optional)
string

The field of the master grid row that the ES00DocumentGrid will be a detail of. The value of this field in the master row will form the parameter for fetchES00DocumentsByGID service to retrieve the ES00DocumentRows.
