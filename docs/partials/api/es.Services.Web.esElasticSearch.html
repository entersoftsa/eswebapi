esElasticSearch
service in module es.Services.Web

Description

esElasticSerach is a facade provided by the ES WEB API in order to facilitate and secure the access to an Elastic Search, registered in the ES WEB API Server through the esWebApi service. All the calls are actually passed through to the registeredto the Entersoft WEB API Server ElasticSearch engine.

For more information about ElasticSearch please visit ElasticSearch and for the complete reference guide visit Reference Guide.

#ES WEB API Server and ElasticSearch In order to be able to use the esElasticSearch service of the Entersoft AgnularJS API, you have first to register to the ES WEB API Server the ElasticSearch engine that will be serve the queries of the service.

Methods

  • searchFree(eUrl, eMethod, eBody)

    This function acts as a facade to the Elastic Search Engine that has been registered with Entersoft WEB API Server. This function allows for almost any kind of operation can be executed by the ElasticSearch engine as an http method (i.e. GET, PUT, DELETE, POST, etc.) searchIndex.

    See also eSearch

    Parameters
    ParamTypeDetails
    eUrlstring

    the last segment of the URL (i.e. without the domain name) that contains the REST api part e.g. twitter/_settings,_mappings (the full CURL type request URL would be http://localhost:9200/twitter/_settings,_mappings). This call is an example from ElasticSearch documentation GetIndex that retrieves the settings and the mappings for the twitter index.

    eMethodstring

    The http method verb (i.e. "GET", "PUT", "DELETE", "POST", etc.). In the example above the verb should be "GET" the index or indices specified by the elasticSearchIndex parameter.

    eBodystring

    The http request payload as required by the ElasticSearch REST call we are executing. In the example above there is no payload so it would be emtpy or null.

    Returns
    httpPromise

    The response that we get from Elastic Search service should we perform an AJAX call against the service

  • searchIndex(elasticSearchIndex, elasticSearchQuery)

    This function acts as a facade to the Elastic Search Engine that has been registered with Entersoft WEB API Server for the Search service based on the elastic search query language. This is the most common scenario of Elastic Search usage i.e. querying the Elastic Search database within the scope of one or more indices. See also searchIndexAndDocument.

    Parameters
    ParamTypeDetails
    elasticSearchIndexstring

    The Elastic Search index name or comma separated index names or reg expression resolving to index - indices to which the elasticSearchQuery statement will be executed against.

    elasticSearchQuerystring

    The Elastic Search valid statement - query to be executed. For more information about ElsticSearch Query Language please visit ES Query DSL.

    Returns
    httpPromise

    The response that we get from Elastic Search service should we perform an AJAX call against the service

  • searchIndexAndDocument(elasticSearchIndex, elasticSearchDocumentType, elasticSearchQuery)

    This function acts as a facade to the Elastic Search Engine that has been registered with Entersoft WEB API Server for the Search service based on the elastic search query language. This is the most common scenario of Elastic Search usage i.e. querying the Elastic Search database within the scope of one or more indices for a limited set of documents of the given Document Types. It is similar to searchIndex.

    Parameters
    ParamTypeDetails
    elasticSearchIndexstring

    The Elastic Search index name or comma separated index names or reg expression resolving to index - indices to which the elasticSearchQuery statement will be executed against.

    elasticSearchDocumentTypestring

    The Elastic Search Document Type or Types (in comma separated list) for which the elastic search query will be executed against within the index or indices specified by the elasticSearchIndex parameter.

    elasticSearchQuerystring

    The Elastic Search valid statement - query to be executed. For more information about ElsticSearch Query Language please visit ES Query DSL.

    Returns
    httpPromise

    The response that we get from Elastic Search service should we perform an AJAX call against the service
