esGrid
directive in module es.Web.UI

Description

TBD This directive is responsible to render the html for the presentation of the results / data of an Entersoft Public Query. The esGrid generates a Telerik kendo-grid web ui element kendo-grid.

In order to instantiate an esGrid with an Angular application, you have to provide the parameters esGroupId and esFilterId are required. These two parameters along with esExecuteParams will be supplied to the esToKendoTransform function

Dependencies

  • esWebApi

    Entersoft AngularJS WEB API for Entersoft Application Server

  • esUIHelper
  • $log

Usage

as element:
<es-grid
       es-group-id="{template}"
       es-filter-id="{template}"
       [es-grid-options="{esGridInfoOptions}"]
       [es-execute-params="{object}"]>
</es-grid>
as attribute
<ANY es-grid
     es-group-id="{template}"
     es-filter-id="{template}"
     [es-grid-options="{esGridInfoOptions}"]
     [es-execute-params="{object}"]>
   ...
</ANY>
