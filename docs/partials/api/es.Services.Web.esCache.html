esCache
service in module es.Services.Web

Description

esCache is a service that provides chachinf functionality to the Entersoft AngularJS API library that can be also used from the application developer for the needs of his/her Single Page Application. Although, the functions offered by the service are not specific to any of the publicly available javascript libraries, the current version of the service relies on jscache, but this can change any time in the future with 100% compatibility to the functions offered by the service i.e. the exposed functions and their signature is an interface to the client developer, but not the implementation.

Methods

  • clear()

    This function clears the cache, by removing ALL its entries.

  • getItem(key)

    This function is used to get a value that might exist in the cache under the unique id key.

    Parameters
    ParamTypeDetails
    keyObjectstring

    Typically key is of type string but it can be an object.

    Returns
    Object

    If the key exists in the cache, the value of that key is returned, otherwise undefined

  • removeItem(key)

    This function is used to remove an entry from the cache identified by key

    Parameters
    ParamTypeDetails
    keyObjectstring

    Typically key is of type string but it can be an object.

  • setItem(key, val, options)

    This function is used to set a key, value pair in the cache. If the key is already in the cache, its value will be replaced by the new value i.e. val

    Parameters
    ParamTypeDetails
    keyObjectstring

    Typically key is of type string but it can be an object.

    valObject

    The value to be stored under the key

    options
    (optional)
    		Object

    Caching options. For more information please visit monsur jscache.

  • stats()

    This function returns cache statistics

    Returns
    object

    It returns an object holding all cache statistics i.e.

    {
    "hits": 5,
    "misses": 1
